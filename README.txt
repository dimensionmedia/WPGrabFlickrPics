=== WPGrabFlickrPics ===
Contributors: dimensionmedia
Donate link: http://davidbisset.com/
Tags: instagram
Requires at least: 3.6
Tested up to: 3.6.1

This plugin will search through recent Flickr posts (containing a certain hashtag or keyword) on the main timeline, and import those photos along with some metadata into WP's media gallery.

== Installation ==

1. Upload the WPGrabFlickrPics folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update a key term in the options, click the "grab" button in the grab area and watch the magic (look in your media area). 

== What Will I Need? ==

For the moment, nothing.
